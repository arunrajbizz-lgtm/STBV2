# POOMANI IPTV

Premium Stalker Portal IPTV application for Samsung Tizen TV and Web.

## Features
- Full Stalker Portal Support
- Multi-Provider Management
- Netflix-style UI
- Live TV with EPG Preview
- Movies & Series (OTT Navigator style)
- Samsung Tizen Remote Navigation
- Premium Fullscreen Player (HTML5 + AVPlay)

## Development
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Deployment
- **Tizen**: Package the `dist` folder with Tizen CLI.
- **Web**: Deploy to Vercel/Railway.
